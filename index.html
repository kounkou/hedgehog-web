About

Hedgehog is a user-friendly tool designed to help users tackle challenges and track their progress efficiently. It features a clean interface and powerful tools to streamline the problem-solving experience.

Key features of Hedgehog include:

  • A timer to keep users focused and engaged.
  • The ability to pause and resume the timer, automatically restarting when the user begins editing again, offering flexibility for interrupted sessions.
  • Language support, currently available for C++ and Golang, with potential for easy expansion to additional languages.
  • Light and dark themes to reduce eye strain and enhance user comfort.
  • A solution display with a toggle to switch between the user's submitted code and the expected solution.
  • The option to skip questions and move on to the next challenge.
  • An option to retry questions, fostering continued learning and improvement.
  • Automatically generated statistics for tracking progress and performance.
